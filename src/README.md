# Sistema de Agenda de Eventos

Nessa atividade utilizei métodos e conceitos que eu já conhecia,
mas também aprendi novas funcionalidades relacionadas à API DateTime do Java.

Achei um assunto interessante até o momento. Atualmente estou estudando:

- LocalDateTime
- DateTimeFormatter
- parse()
- ZonedDateTime
- Period
- manipulação de datas e horários

Um ponto que achei interessante no projeto foi o uso do `contains()`
para permitir buscas parciais no nome dos eventos.

## Conceitos utilizados no projeto

- LocalDateTime
- DateTimeFormatter
- parse()
- ArrayList
- for
- for each
- métodos
- classes
- encapsulamento
- remoção de elementos
- busca parcial com contains()

## Funcionalidades

- Cadastro de eventos
- Listagem de eventos
- Remoção de eventos
- Busca parcial por nome
